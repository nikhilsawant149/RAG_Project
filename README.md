# FEM RAG Assistant

A Retrieval-Augmented Generation (RAG) chatbot for learning the Finite Element Method (FEM).

## Features

- Streamlit UI
- Chroma Vector Database
- OpenAI Embeddings
- Mistral LLM
- Conversation Memory
- Source Citation

## Tech Stack

- Python
- LangChain
- Chroma
- Streamlit
- Mistral AI
- OpenAI Embeddings

## Installation

```bash
uv venv
```

Activate

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
uv pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
